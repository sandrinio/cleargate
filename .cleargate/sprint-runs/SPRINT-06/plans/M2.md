### Diagnosis (cite)

`.claude/hooks/token-ledger.sh:45` reads `transcript_path` from stdin ‚Äî **Claude Code's SubagentStop payload always carries the ORCHESTRATOR session's `.jsonl`**, not a per-subagent transcript. ### Fix: option (a) per-Task sentinel (lowest-friction)

Option (b) per-invocation transcript scan is infeasible ‚Äî Claude Code does not emit subagent transcripts as separate files (verified above). Option (a) is the only path.

**Mechanism:**
- Orchestrator writes `.cleargate/sprint-runs/<sprint>/.pending-task-<N>.json` **before each Task dispatch** with `{agent_type, work_item_id, started_at, turn_index}` where `turn_index` = count of assistant turns in the orchestrator transcript at dispatch time. The orchestrator's dispatch prompt already leads with `STORY=NNN-NN` ‚Äî the human orchestrator (me) owns writing this sentinel as part of the dispatch protocol. No Claude-Code SDK change.
- SubagentStop hook enhancement: on fire, find the newest `.pending-task-*.json` under the active sprint dir; compute token usage as the **delta** between `turn_index` and the orchestrator transcript's current assistant-turn count (instead of summing the entire transcript each fire ‚Äî which is why SPRINT-05 rows have ballooning `input`/`cache_read` fields). Attribute the row with the sentinel's `{agent_type, work_item_id}`. Delete the sentinel.

**Why the delta model:** current code (`token-ledger.sh:73-82`) sums ALL assistant turns in the transcript every fire. SPRINT-05's 61 rows therefore each double-count the prior rows' usage. Reporter can't reconstruct per-agent cost from cumulative sums without subtraction ‚Äî the delta model records the actual per-subagent slice.

**Fallback** when no sentinel exists (direct orchestrator work, no Task dispatch): write to `.pending-task-orchestrator.json` on session start (via SessionStart hook ‚Äî already wired at `.claude/settings.json:24-33`), delta against that. Agent_type = `orchestrator`.

  ```ts
  // Module-level state (NOT localStorage/sessionStorage ‚Äî story ¬ß4 DoD)
  let adminToken: string | null = null;
  let expiresAt: Date | null = null;
  let refreshTimer: ReturnType<typeof setTimeout> | null = null;

  export async function exchange(): Promise<{admin_token: string; expires_at: string}> {
    const res = await fetch(`${PUBLIC_MCP_URL}/admin-api/v1/auth/exchange`, {
      method: 'POST', credentials: 'include'
    });
    if (res.status === 401) throw new AuthError('session_expired');
    if (res.status === 403) throw new ForbiddenError('not_authorized');
    if (!res.ok) throw new NetworkError(res.status);
    const body = AuthExchangeResponseSchema.parse(await res.json());
    adminToken = body.admin_token; expiresAt = new Date(body.expires_at);
    scheduleRefresh(expiresAt); return body;
  }

  export async function get<T>(path: string, schema: ZodSchema<T>): Promise<T> {
    if (!adminToken) await exchange();
    let res = await fetch(`${PUBLIC_MCP_URL}/admin-api/v1${path}`,
      { headers: { Authorization: `Bearer ${adminToken}` }});
    if (res.status === 401) {
      await exchange();
      res = await fetch(`${PUBLIC_MCP_URL}/admin-api/v1${path}`,
        { headers: { Authorization: `Bearer ${adminToken}` }});
      if (res.status === 401) throw new AuthError('session_expired');
    }
    if (res.status === 403) throw new ForbiddenError('forbidden');
    if (!res.ok) throw new NetworkError(res.status);
    return schema.parse(await res.json());
  }

  function scheduleRefresh(at: Date) {
    if (refreshTimer) clearTimeout(refreshTimer);
    const ms = at.getTime() - Date.now() - 2 * 60 * 1000; // 2 min before expiry
    if (ms > 0) refreshTimer = setTimeout(() => exchange().catch(() => {}), ms);
  }

  export function signOut() {
    adminToken = null; expiresAt = null;
    if (refreshTimer) { clearTimeout(refreshTimer); refreshTimer = null; }
  }
  ```
