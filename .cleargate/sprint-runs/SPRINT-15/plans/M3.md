**Files to create:**

  - Exports:
    ```ts
    export interface LedgerRowV2 {
      ts: string;
      sprint_id: string;
      story_id: string;
      work_item_id: string;
      agent_type: string;
      session_id: string;
      delta: { input: number; output: number; cache_creation: number; cache_read: number };
      session_total: { input: number; output: number; cache_creation: number; cache_read: number };
      model: string;
      turns: number;
    }

    export interface SumResult {
      totals: { input: number; output: number; cache_creation: number; cache_read: number };
      format: 'delta' | 'pre-0.9.0' | 'mixed';
      pre_v2_caveat?: string;
    }

    export function sumDeltas(rows: unknown[]): SumResult;
    ```
  - Format detection algorithm (mandatory contract):
    1. For each row, check presence of `row.delta` object with all four numeric fields.
    2. If **all** rows have `delta` ‚Üí `format: 'delta'`, sum `delta.*` directly.
    3. If **no** rows have `delta` ‚Üí `format: 'pre-0.9.0'`, group by `session_id`, take the row with max `ts` per session, sum its top-level `input/output/cache_*` fields (the "last-row trick").
    4. If **some** rows have `delta` and others don't ‚Üí `format: 'mixed'`. Walk the ledger in `ts` order; for each session_id, switch behavior based on the format of the row at that offset: delta-rows contribute their `delta.*`; flat-rows for the same session use the last-row trick **scoped to flat-rows only** within that session. Emit `pre_v2_caveat: "Mixed format ledger: <N> delta rows + <M> pre-0.9.0 rows; flat segment uses last-row trick."` Reporter pastes this verbatim into REPORT.md ¬ß3.
  - **Mixed-format motivation:** SPRINT-15 itself ships across the cutover. Rows from CR-018-pre commits are flat; rows from CR-018-post commits carry `delta`. Reporter must NOT crash on this hybrid.

**Schema ‚Äî new ledger row (verbatim, append after CR-018 ships; pre-existing rows retain old shape):**

**Reporter math (exact additive operation):**

| Old (pre-CR-018) | New (CR-018+) |
|---|---|
| `sum(rows[i].input + rows[i].output + rows[i].cache_creation + rows[i].cache_read)` for all i ‚Äî multiplicative inflation √ó N fires per session | `sum(rows[i].delta.input + rows[i].delta.output + rows[i].delta.cache_creation + rows[i].delta.cache_read)` for all i, regardless of session boundaries ‚Äî additive across multi-session sprints |
