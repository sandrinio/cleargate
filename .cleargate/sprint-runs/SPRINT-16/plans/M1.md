### STORY-016-01 — Registry-check library + 24h cache
- **Files to create:** `src/lib/registry-check.ts`, `test/lib/registry-check.test.ts`.
- **API contract (LOCKED — 016-02 consumes verbatim):**
  ```ts
  export type CheckSource = 'cache' | 'network' | 'opt-out' | 'error';
  export interface CheckResult { latest: string | null; from: CheckSource; }
  export interface CheckOptions {
    fetcher?: typeof fetch;       // test-seam injection
    now?: () => Date;             // test-seam for 24h throttle
    cleargateHome?: string;       // test-seam for ~/.cleargate (mirrors doctor.ts seam at line 38)
    env?: NodeJS.ProcessEnv;      // test-seam for CLEARGATE_NO_UPDATE_CHECK
  }
  export async function checkLatestVersion(opts?: CheckOptions): Promise<CheckResult>;
  ```
- **Cache file:** `path.join(cleargateHome ?? path.join(os.homedir(), '.cleargate'), 'update-check.json')`. Shape: `{ checked_at: <ISO8601>, latest_version: <string> }`. mkdir parent recursively before write.
- **Network:** `globalThis.fetch('https://registry.npmjs.org/cleargate', { signal: AbortSignal.timeout(2000) })`. Parse only `body['dist-tags'].latest`; nothing else.
- **Failure paths:** any throw OR non-2xx OR malformed JSON → return `{ latest: <cached_or_null>, from: 'error' }`. Never throw, never write to stderr, never console.error.
- **Opt-out:** if `(opts?.env ?? process.env).CLEARGATE_NO_UPDATE_CHECK === '1'` return `{ latest: null, from: 'opt-out' }` immediately — do not read cache, do not write cache.
- **Throttle:** read cache; if `Date.now() - Date.parse(checked_at) < 24*60*60*1000` return `{ latest, from: 'cache' }` and skip the network call entirely.
- **Test scenarios (6, one per Gherkin):**
  1. Fresh check writes cache, returns `{from:'network'}`.
  2. <24h cache short-circuits network (assert mock fetcher never called).
  3. >24h cache triggers refresh.
  4. Opt-out env returns `{latest:null,from:'opt-out'}`, no cache write.
  5. Network rejection with stale cache returns `{latest:cached,from:'error'}`, no throw, no stderr.
  6. Network rejection without cache returns `{latest:null,from:'error'}`.
- **Reuse (no duplication):** None — no existing registry-fetch helper. Test seam pattern mirrors `doctor.ts:38` (`cleargateHome` injection for `~/.cleargate`).
- **Gotchas:**
  - `AbortSignal.timeout` is Node 20+ — repo engines is `>=24.0.0` (package.json:42), safe.
  - Capture `process.env` at call-time, not module-load — tests mutate env between cases.
  - The test for Scenario 4 must assert **no cache file write** by stat-ing the cache path after the call. A vi.spyOn wrapping `fs.writeFileSync` catches false positives where the implementation skips the network but still touches the cache.
